The canvas size is 800 x 500. (0,0) is in the top left, (800, 500) in the bottom right. All objects are added to an object list. The animation function works in the following steps:

  1. The canvas is cleared of the previous drawing.
  2. Each objects collision list is cleared.
  3. Each object determines if it is colliding with another object and if it is, it calculates the redirection for both it and the object it collides with. Once objects have collided, they both add each other to their collision list (I'm not sure if this is needed).
  4. Each objects next position is calculated.
  5. Each objects is redrawn on the screen.
